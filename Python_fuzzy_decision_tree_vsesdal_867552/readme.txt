Необходимо написать программу на языке Питон 3, которая осуществляет построение нечетких деревьев решений. 
Входными данными программы являются два файла: обучающая выборка(та выборка, по которой строится дерево) и тестовая выборка(выборка, на которой проверяется точность дерева). 
Результирующим является 2 файла - 1й файл содержит построенное визулизированное дерево, а 2й файл содержит тестовую выборку, с, спрогнозированнными по дереву, классами, а также с значением точности построенного дерева. Данные в выборках могут быть как дискретными, так и непрерывными. Построение нечеткого дерева основано на алгоритме ID3, файл прилагается (алгоритм описан на страницах 16 - 25)
